# Afterglow Frontend (Vue)

Afterglow 前端初始工程，技术栈：
- Vue 3
- Vite
- Vue Router
- Pinia

## 本地运行

```bash
npm install
npm run dev
```

默认地址：`http://localhost:5173`

## 环境变量

复制 `.env.example` 为 `.env`（可选）：

```bash
VITE_API_BASE_URL=http://localhost:8080/api
```

## 当前页面

- `/`：项目启动页 + 后端健康检查联调
- `/roadmap`：开发路线初稿
